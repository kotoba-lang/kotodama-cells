# yorishiro_demo-fixture

Pregel cell for the **demo-fixture** yorishiro (kami: `bin:demo-fixture`,
binary: `demo-fixture`).

Per ADR-2605211900 + ADR-2605202200. Generated by `@etzhayyim/yorishiro`
v0.1.0 in **binary-cli** mode.

## Ops

| Op | Summary |
|---|---|
| `greet` | Print a greeting for NAME. |
| `head` | Read up to MAX_LINES from INPUT_PATH and write to OUTPUT_PATH. |

## Runtime requirements

The `demo-fixture` binary must be available on the
cell runtime's PATH (or supplied as an absolute path in the kami
manifest at `00-contracts/kami/demo-fixture.kami.json`). The cell
verifies this via `shutil.which()` at the start of every invocation
and returns `error: "binary not found on PATH"` if missing.

intend the MCP server to dispatch to a remote runtime. For the simple

on the host's PATH.
